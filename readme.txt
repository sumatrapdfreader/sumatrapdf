SumatraPDF is a multi-format (PDF, EPUB, MOBI, FB2, CHM, XPS, DjVu) reader
for Windows under (A)GPLv3 license, with some code under BSD license (see
AUTHORS).

The main source repository is at http://code.google.com/p/sumatrapdf/ ,
a read-only git mirror is at https://github.com/kjk/sumatrapdf .

Want to join the project and contribute code? Read
https://code.google.com/p/sumatrapdf/wiki/JoinTheProject

To compile you need Visual Studio 2013 (Express).
Either open the right solution from vs directory or call nmake from a
Visual Studio command prompt. For creating (pre)release builds and the
installer, call scripts/build-release.bat (requires Python).
